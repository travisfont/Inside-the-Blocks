# Tokens

## Asset-Referenced Tokens (ART)
> Also known as "asset-backed", these tokens are aim to keep a stable value by referencing a basket of assets, such as multiple currencies, commodities, or other assets.

### Fiat-backed Stablecoin
Backed by traditional money or cash-like assets (bank deposits, short-term U.S. Treasuries).
- USDC
- USDT

### Commodity-Backed Token
Backed by physical commodities stored by a custodian.
- PAXG (gold-backed)

### Crypto-Collateralized Token
Backed by other cryptocurrencies, often created through overcollateralized loans.
- DAI

### Real-World Asset (RWA) Token
Backed by things like bonds, invoices, real estate shares, or funds, represented on-chain.

## E-Money Token (EMT)
> Crypto-assets that aim to keep a stable value by referencing one single official currency (like EUR or USD).

### Algorithmic Stablecoin

## Utility Token
> Everything that is not an ART or an EMT which includes utility tokens, which are tokens mainly used to access a product/service offered by the issuer.
